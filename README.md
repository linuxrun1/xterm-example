# Xterm.js Example

使用 [xterm.js](https://github.com/sourcelair/xterm.js) 类库结合 Node.js 完成 web terminal 例子。

## Dependence

* Node.js v8+
* npm v5+

## Install

```bash
git clone https://github.com/lovefishs/xterm-example.git
cd xterm-example
npm install
```

## Dev

```bash
npm run dev
```

Open a new command line

```bash
node server/app.js
```

## Build

```bash
npm run build
```

## Start

```bash
node server/app.js
```
